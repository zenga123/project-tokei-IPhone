# Project Tokei
トケイプロジェクトは、24時間のアナログ時計UIを通じて、ユーザーが一目でその日のスケジュールを把握できるよう、直感的な視覚化を提供することを目標に制作されました。
また、AI分析機能を活用して効率的な時間管理と最適なスケジュール提案を提供することで、
ユーザーに有益なスケジュール管理体験を実現することを目指しております。
